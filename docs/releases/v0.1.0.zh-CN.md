# 发布说明 v0.1.0

`v0.1.0` 是 CogniWeave Memory 以 Python 包名 `cogni-mem` 对外公开发布的首个版本。

## 本版重点

- 已发布到 PyPI，安装名为 `cogni-mem==0.1.0`
- 基于内部架构阶段 `范式+记忆系统&RAG重构 v0.2`
- 保持框架核心行为与该源码里程碑一致
- 通过 `pyproject.toml` 补齐标准 Python 打包能力
- 将可选能力拆分为 extras，而不是把所有集成都塞进基础安装
- 补齐双语 README 与架构说明文档
- 采用 MIT License 开源

## 安装

```bash
pip install cogni-mem==0.1.0
```

可选 extras：

```bash
pip install "cogni-mem[minimax]==0.1.0"
pip install "cogni-mem[graph]==0.1.0"
```

从 GitHub 标签安装：

```bash
pip install "git+https://github.com/luckly06/cogniweave-memory.git@v0.1.0"
```

## 发布范围

这一版是明确的“打包发布版”，不主动改动框架核心实现行为。主要工作集中在公开分发所需的元数据、依赖分层、许可证和文档整理。

## 运行模型

- 默认本地运行模式：
  key memory 使用 JSON，metadata 使用 SQLite，vector memory 使用本地 Qdrant path 模式
- 可选远程 Qdrant：
  通过 `COGNIWEAVE_ENABLE_QDRANT=true` 启用
- 可选 Neo4j 图集成：
  安装 `graph` extra，并启用 `COGNIWEAVE_ENABLE_NEO4J=true`

## 验证项
